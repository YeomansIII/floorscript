## [0.1.0] - 2025-01-01

### Added

- Initial release
- YAML/JSON floor plan configuration parsing with Zod validation
- Layout resolver with room positioning, wall generation, and opening placement
- SVG rendering engine with architectural drawing conventions
- Dimension lines (auto-generated and manual)
- Electrical system support (panels, outlets, switches, circuits)
- Plumbing system support (fixtures, supply runs, drain runs)
- Multi-layer rendering (structure, electrical, plumbing, dimensions)
- Imperial and metric unit systems
- CLI with `render` and `init` commands
